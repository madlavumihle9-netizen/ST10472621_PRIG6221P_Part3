## 📋 Project Overview

A WPF-based cybersecurity awareness chatbot built in C# with .NET 8.0. The application educates users on online safety through interactive conversations, task management, quizzes, and activity tracking.

### Part 1 (Console App — 78%)
- Console-based chatbot with ASCII art, voice greeting, and 8 cybersecurity topics
- Randomized responses, sentiment detection, and follow-up questions

### Part 2 (WPF GUI — 90%)
- Dark cybersecurity-themed GUI (navy/cyan/white)
- All Part 1 features preserved with professional layout
- 10 cybersecurity keywords, 5 sentiment moods, memory/recall system

### Part 3 (Final — Current)
- **Task Assistant:** Add, view, complete, and delete cybersecurity tasks with JSON storage
- **Quiz Mini-Game:** 12 questions (multiple choice + true/false) with immediate feedback
- **NLP Simulation:** Detects varied phrasing for tasks, reminders, quiz, and log intents
- **Activity Log:** Tracks all bot actions with timestamps, shows 5-10 entries at a time
- **Splash Screen:** Animated welcome screen with name capture and voice greeting
